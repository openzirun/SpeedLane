# 参与贡献

感谢你对 SpeedLane 的兴趣!

## 开发环境

- macOS 13+,Xcode / Swift 5.9+
- 克隆后直接 `swift build` 即可编译,`./build_app.sh` 打包成 .app

## 提交 PR

1. Fork 并创建特性分支
2. 保持代码风格与现有代码一致(中文注释,SwiftUI + AppKit)
3. 确认 `swift build -c release` 通过
4. PR 描述里说明改动动机和测试方式

## 报告问题

提 Issue 时请附上:

- macOS 版本
- 复现步骤
- 连接失败类问题请附"设置 → 服务器 → 测试连接"的错误信息
